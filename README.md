# RPN计算器 - C++编程作业

## 项目简介
这是一个基于命令行界面的逆波兰表示法（RPN）计算器，完全符合作业要求。

## 功能特性

### 基础功能（已实现）
✅ **四则运算**：`+` `-` `*` `/`
✅ **栈操作**：数字输入、清空栈、显示栈
✅ **连续运行**：支持完整的RPN表达式
✅ **错误处理**：除零、格式错误、栈空等

### 高级功能（已实现）
✅ **数学运算**：平方根(`sqrt`)、幂运算(`pow`)
✅ **特殊功能**：斐波那契数列(`fib`)
✅ **历史记录**：记录所有计算操作
✅ **三角函数**：正弦(`sin`)、余弦(`cos`)
